What's new in REST API Version 23.0

Edit online

IBM® QRadar® 7.5.0 Update Package 9 introduces version 23.0 of the API endpoints.

New endpoints

RESTful API 23.0 introduces new categories of API endpoints and updates to existing endpoints in the following categories:
  • Configuration Endpoints
  • Event Sources
  • WinCollect
All version 20.x API endpoints are marked as deprecated in QRadar 7.5.0 Update Package 8.
Note: Although deprecated endpoints continue to function, they will be removed in a future release. You must update your integration to use the most recent version of the QRadar RESTful API. Responses to deprecated endpoint requests include a Deprecated response header.
